# 数学建模协会网站 - GitHub Pages 部署指南

## 部署步骤

### 步骤1: 创建GitHub仓库

1. 打开 https://github.com ，登录你的账号
2. 点击右上角 **"+"** → **"New repository"**
3. 填写:
   - Repository name: `math-modeling-association` (或其他名字)
   - Description: `武昌工学院数学建模协会官方网站`
4. 选择 **"Public"**
5. 点击 **"Create repository"**

### 步骤2: 推送代码

创建仓库后，页面会跳转。在页面下方找到 "…or push an existing repository from the command line"，复制下面的命令执行：

```
cd "C:\Users\24295\Desktop\协会网站"
git branch -M main
git remote add origin https://github.com/你的用户名/math-modeling-association.git
git push -u origin main
```

### 步骤3: 开启GitHub Pages

1. 进入仓库页面
2. 点击 **Settings** (设置)
3. 左侧找到 **"Pages"**
4. 在 "Source" 下选择 **"Deploy from a branch"**
5. Branch 选择 **"main"**，文件夹选择 **"/(root)"**
6. 点击 **"Save"**

### 步骤4: 获取网址

等待1-2分钟后，刷新页面，顶部会出现类似:
```
Your site is live at https://你的用户名.github.io/math-modeling-association/
```

## 常见问题

**Q: 如何更新网站内容？**
A: 修改 `index.html` 后，执行:
```bash
git add .
git commit -m "更新内容"
git push
```

**Q: 网址太长了？**
A: 可以买一个短域名（如 mathclub.cn），在 Pages 设置里绑定。
